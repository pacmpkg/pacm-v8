# Support

Thanks for using pacm-v8! This document outlines the best ways to get help.

## Documentation

- Crate documentation: <https://docs.rs/pacm-v8>
- Repository README: [README.md](README.md)
- Examples: `example/`

## Questions

- Use GitHub Discussions or the `question` issue template if you have how-to or troubleshooting questions.
- For pacm-specific topics, consider reaching out in the broader pacm community channels.

## Bugs and Feature Requests

- File issues via [GitHub Issues](https://github.com/pacmpkg/pacm-v8/issues)
- Include reproduction steps, logs, and minimal examples where possible.

## Security Concerns

- Follow the private disclosure process in [SECURITY.md](SECURITY.md).
